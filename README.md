## 🛠️ Technologies

- **Java** 21
- **Spring Boot** 3.x
- **Maven**
- **MySQL/PostgreSQL**

## 🚀 How to Run

```bash
# Run each lesson
cd Lesson01/Lesson01
mvn spring-boot:run

# Run project
cd Project/BE
mvn spring-boot:run
```

## 📋 Requirements

- JDK 21+
- Maven 3.8+
- IDE (IntelliJ IDEA, VS Code,...)
